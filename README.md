# Data Visualization and Storytelling Task

## Objective
Analyzed the Superstore dataset to uncover profit trends and identify loss-making categories.

## Tools Used
- Google Colab / Python (Pandas, Seaborn)
- Visualization: Matplotlib

## Key Insights
- Identified that the 'Furniture' category is the primary driver of loss.
- Total sales show a positive growth trend, but profit margins remain unstable.
